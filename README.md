## development
* install hardhat
  ```
  cd hardhat
  yarn install
  ```
* install ui dependency
  ```
  cd ui
  yarn install
  ```
* start hardhat
  ```
  make hh-node
  ``` 
* start server
  ```
  make start-server
  ```
* start ui
  ```
  make start-ui
  ```
* visit: http://localhost:8080
